## System Modules

### system/system_update

Updates macOS and installs core components.

**Actions:**
- Runs `softwareupdate` for macOS updates
- Installs Xcode Command Line Tools
- Installs Rosetta 2 (Apple Silicon only)

**Requirements:** None (run first)

---

### system/directories

Creates directories and configures Time Machine.

**Actions:**
- Creates directories from `DIRS` array
- Adds Time Machine exclusions from `DIRSTOEXCLUDEFROMTIMEMACHINE`

**Config:**
```bash
DIRS=("~/GIT" "~/Projects")
DIRSTOEXCLUDEFROMTIMEMACHINE=("~/GIT" "~/.npm")
```

---

### system/security

Hardens macOS security settings.

**Actions:**
- Enables FileVault disk encryption
- Enables application firewall (stealth mode)
- Enables Gatekeeper
- Disables guest user
- Requires password after sleep
- Enables secure keyboard entry

**Requirements:** Sudo access

---

### system/cleanup

Cleanup tasks and app restarts.

**Actions:**
- Clears system caches
- Restarts apps from `KILLAPPS` array

**Config:**
```bash
KILLAPPS=("Dock" "Finder" "SystemUIServer")
```

---

## Application Modules

### applications/homebrew

Installs Homebrew and packages.

**Actions:**
- Installs Homebrew if missing
- Adds taps from `BREWTAPS`
- Installs packages from `BREWPACKAGES`
- Installs casks from `BREWCASKS`

**Config:**
```bash
BREWTAPS=("homebrew/cask")
BREWPACKAGES=("git" "jq")
BREWCASKS=("visual-studio-code")
```

---

### applications/apps

Installs Mac App Store apps.

**Actions:**
- Installs apps from `APPSTORE` array via `mas`

**Config:**
```bash
APPSTORE=("441258766" "937984704")  # Magnet, Amphetamine
```

**Requirements:** `applications/homebrew` (installs `mas`)

---

### applications/espanso

Installs Espanso text expander.

**Actions:**
- Installs Espanso
- Configures extensions

---

## Development Environment Modules

### dev_environments/python

Sets up Python via pyenv.

**Actions:**
- Installs pyenv, pyenv-virtualenv
- Installs Python version from `PYTHONVER`
- Sets global default
- Installs packages from `GLOBAL_PYTHON_PACKAGES`

**Config:**
```bash
PYTHONVER="3.11.0"
GLOBAL_PYTHON_PACKAGES=("pip" "virtualenv" "black")
```

---

### dev_environments/node

Sets up Node.js via nodenv.

**Actions:**
- Installs nodenv
- Installs Node.js version from `NODEVER`
- Installs packages from `GLOBAL_NODE_PACKAGES`

**Config:**
```bash
NODEVER="20.10.0"
GLOBAL_NODE_PACKAGES=("eslint" "prettier")
```

---

### dev_environments/ruby

Sets up Ruby via rbenv.

**Config:** `RUBYVER="3.2.0"`

---

### dev_environments/go

Sets up Go via goenv.

**Config:** `GOVER="1.21.0"`

---

### dev_environments/rust

Sets up Rust via rustup.

**Actions:**
- Installs rustup
- Installs stable toolchain
- Adds rustfmt, clippy

---

### dev_environments/java

Sets up Java via jenv.

**Actions:**
- Installs jenv
- Installs Temurin JDK from `JDKVER`
- Installs Maven

**Config:** `JDKVER="21"`

---

### dev_environments/php

Sets up PHP and Composer.

**Config:**
```bash
PHPVER="8.3.0"
GLOBAL_PHP_PACKAGES=("phpunit/phpunit")
```

---

### dev_environments/flutter

Sets up Flutter SDK.

**Config:** `FLUTTER_CHANNEL="stable"`

---

## IDE Modules

### ide/ide_setup

Orchestrates IDE installations based on config flags.

**Config:**
```bash
INSTALL_VSCODE=true
INSTALL_CURSOR=false
INSTALL_JETBRAINS=false
INSTALL_SUBLIME=false
INSTALL_ZED=false
```

---

### ide/vscode_setup

Installs VS Code and extensions.

**Triggered by:** `INSTALL_VSCODE=true`

---

### ide/jetbrains_setup

Installs JetBrains IDEs.

**Triggered by:** `INSTALL_JETBRAINS=true`

---

### ide/sublime_setup

Installs Sublime Text and packages.

**Triggered by:** `INSTALL_SUBLIME=true`

---

### ide/zed_setup

Installs Zed editor.

**Triggered by:** `INSTALL_ZED=true`

---
