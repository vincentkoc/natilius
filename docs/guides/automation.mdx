Natilius is built for infrastructure-as-code workflows.

## Environment Variables

Control Natilius behavior without editing config files:

| Variable | Default | Description |
|----------|---------|-------------|
| `SKIP_SUDO` | `false` | Skip sudo prompts (CI/CD) |
| `NONINTERACTIVE` | `false` | No prompts at all |
| `DRY_RUN` | `false` | Preview mode |
| `QUIET_MODE` | `false` | Errors only |
| `VERBOSE_MODE` | `false` | Debug output |

### Fully Automated Run

```bash
SKIP_SUDO=true \
NONINTERACTIVE=true \
QUIET_MODE=true \
natilius setup
```

## Terraform

### Using the Provisioning Script

```hcl
resource "null_resource" "mac_setup" {
  connection {
    type        = "ssh"
    host        = var.mac_host
    user        = var.mac_user
    private_key = file(var.ssh_key_path)
  }

  provisioner "remote-exec" {
    inline = [
      "curl -fsSL https://raw.githubusercontent.com/vincentkoc/natilius/main/scripts/terraform-provision.sh | bash -s ${var.profile}"
    ]
  }
}

variable "profile" {
  default = "devops"
}
```

### Custom Configuration

```hcl
resource "null_resource" "mac_setup" {
  connection {
    type = "ssh"
    host = var.mac_host
    user = var.mac_user
  }

  # Upload config
  provisioner "file" {
    source      = "configs/${var.role}.natiliusrc"
    destination = "~/.natiliusrc"
  }

  # Install and run
  provisioner "remote-exec" {
    inline = [
      "curl -fsSL https://raw.githubusercontent.com/vincentkoc/natilius/main/install.sh | SKIP_RUN=true bash",
      "NONINTERACTIVE=true ~/.natilius/natilius.sh setup"
    ]
  }
}
```

## Ansible

```yaml
---
- name: Setup Mac Development Environment
  hosts: macs
  become: no

  vars:
    natilius_profile: devops

  tasks:
    - name: Install Natilius
      shell: |
        curl -fsSL https://raw.githubusercontent.com/vincentkoc/natilius/main/install.sh | SKIP_RUN=true bash
      args:
        creates: ~/.natilius/natilius.sh

    - name: Copy profile
      copy:
        src: "profiles/{{ natilius_profile }}.natiliusrc"
        dest: ~/.natiliusrc

    - name: Run setup
      shell: NONINTERACTIVE=true ~/.natilius/natilius.sh setup
      environment:
        PATH: "/opt/homebrew/bin:/usr/local/bin:{{ ansible_env.PATH }}"
```

## GitHub Actions

```yaml
name: Mac Setup

on:
  workflow_dispatch:
    inputs:
      profile:
        description: 'Profile'
        default: 'minimal'
        type: choice
        options: [minimal, devops, developer]

jobs:
  setup:
    runs-on: macos-latest
    steps:
      - name: Install Natilius
        run: |
          curl -fsSL https://raw.githubusercontent.com/vincentkoc/natilius/main/install.sh | SKIP_RUN=true bash

      - name: Run setup
        run: |
          SKIP_SUDO=true NONINTERACTIVE=true ~/.natilius/natilius.sh --profile ${{ inputs.profile }} setup
```

## Idempotency

Natilius is idempotent—safe to run multiple times:

```bash
# First run: installs everything
natilius setup

# Second run: skips installed items
natilius setup  # Much faster
```

### In Terraform

```hcl
resource "null_resource" "mac_setup" {
  triggers = {
    config_hash = filemd5("configs/devops.natiliusrc")
  }

  provisioner "remote-exec" {
    inline = ["~/.natilius/natilius.sh setup"]
  }
}
```

## Logging

Logs are stored at:
```
~/.natilius/logs/natilius-setup-YYYYMMDD-HHMMSS.log
```

Capture output:
```bash
natilius setup 2>&1 | tee /var/log/natilius.log
```
