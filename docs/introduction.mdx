## Why Natilius?

Setting up a new Mac for development is tedious:

- Installing Homebrew and packages
- Configuring Python, Node.js, Ruby, Go...
- Setting up IDEs and extensions
- Tweaking system preferences
- Hardening security settings

**It takes hours.** Natilius does it in minutes.

```bash
curl -fsSL https://raw.githubusercontent.com/vincentkoc/natilius/main/install.sh | bash
natilius setup
```

## System Requirements

| Requirement | Details |

| **macOS** | 13 (Ventura) or later |

| **Disk Space** | ~10GB for full install |
| **Internet** | Required for initial setup |
