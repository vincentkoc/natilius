Complete reference for all `~/.natiliusrc` configuration variables.

## Core Settings

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `ENABLED_MODULES` | array | `()` | Modules to run during setup |
| `COUNTRYCODE` | string | `"us"` | Country code for regional settings |
| `SKIP_UPDATE_CHECK` | bool | `false` | Skip Natilius version check |

## Language Versions

| Variable | Type | Example | Description |
|----------|------|---------|-------------|
| `PYTHONVER` | string | `"3.11.0"` | Python version (via pyenv) |
| `NODEVER` | string | `"20.10.0"` | Node.js version (via nodenv) |
| `RUBYVER` | string | `"3.2.0"` | Ruby version (via rbenv) |
| `GOVER` | string | `"1.21.0"` | Go version (via goenv) |
| `JDKVER` | string | `"21"` | Java version (via jenv) |
| `PHPVER` | string | `"8.3.0"` | PHP version |
| `FLUTTER_CHANNEL` | string | `"stable"` | Flutter channel (stable/beta/dev) |

<Note>
  Version strings must be full semantic versions: `"3.11.0"` not `"3.11"`.
</Note>

## IDE Configuration

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `INSTALL_VSCODE` | bool | `true` | Install Visual Studio Code |
| `INSTALL_CURSOR` | bool | `false` | Install Cursor editor |
| `INSTALL_JETBRAINS` | bool | `false` | Install JetBrains IDEs |
| `INSTALL_SUBLIME` | bool | `false` | Install Sublime Text |
| `INSTALL_ZED` | bool | `false` | Install Zed editor |

## Directories

### DIRS

Directories to create during setup:

```bash
DIRS=(
    "~/.config"
    "~/GIT"
    "~/Projects"
)
```

### DIRSTOEXCLUDEFROMTIMEMACHINE

Directories to exclude from Time Machine backups:

```bash
DIRSTOEXCLUDEFROMTIMEMACHINE=(
    "~/GIT"
    "~/.npm"
    "~/.cargo"
    "~/Library/Caches"
)
```

## Homebrew

### BREWTAPS

Homebrew taps to add:

```bash
BREWTAPS=(
    "homebrew/cask"
    "homebrew/cask-fonts"
    "github/gh"
)
```

### BREWPACKAGES

CLI tools to install:

```bash
BREWPACKAGES=(
    "git"
    "jq"
    "fzf"
    "kubectl"
    "terraform"
)
```

### BREWCASKS

GUI applications to install:

```bash
BREWCASKS=(
    "visual-studio-code"
    "docker"
    "1password"
    "slack"
)
```

## Mac App Store

### APPSTORE

App IDs for Mac App Store apps:

```bash
APPSTORE=(
    "441258766"   # Magnet
    "937984704"   # Amphetamine
)
```

<Tip>
  Find app IDs with `mas search "app name"`.
</Tip>

## Environment Variables

These can be set at runtime:

| Variable | Default | Description |
|----------|---------|-------------|
| `NATILIUS_HOME` | `~/.natilius` | Installation directory |
| `SKIP_SUDO` | `false` | Skip sudo validation |
| `DRY_RUN` | `false` | Preview mode |
| `VERBOSE_MODE` | `false` | Detailed output |
| `QUIET_MODE` | `false` | Minimal output |
| `NONINTERACTIVE` | `false` | No prompts |

Example:
```bash
SKIP_SUDO=true DRY_RUN=true natilius setup
```
